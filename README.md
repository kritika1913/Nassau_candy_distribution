# Factory-to-Customer Shipping Route Efficiency Analysis

## Project Overview

This project analyzes factory-to-customer shipping route efficiency for Nassau Candy Distributor using data analytics and business intelligence techniques.

The objective of the project is to identify shipping delays, route bottlenecks, regional inefficiencies, and ship mode performance using SQL, Power BI, Python, and Streamlit dashboards.

---

## Technologies Used

- Excel
- SQL
- Power BI
- Python
- Streamlit
- Pandas
- Plotly Express

---

## Project Objectives

- Analyze shipping lead times
- Evaluate route efficiency
- Compare ship mode performance
- Detect geographic bottlenecks
- Build interactive dashboards
- Generate operational insights

## Key Insights

- Some regions experienced higher shipping lead times.
- Standard shipping showed more delays compared to expedited shipping.
- Certain states had operational bottlenecks.
- High shipment volume impacted delivery performance.

- Improve monitoring of delayed regions

- Implement predictive logistics analytics
